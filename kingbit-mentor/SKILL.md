# Kingbit Mentor - OpenClaw Skill

## Overview

Your wise mentor who gives profound advice based on time of day and situation. Like having a wise grandfather/mentor who always has the right wisdom for every moment.

## Features

| Feature | Description |
|---------|-------------|
| 🌅 **Morning Wisdom** | Motivational morning advice |
| 🌙 **Evening Wisdom** | Reflective evening thoughts |
| 🎯 **Situation Advice** | Context-aware guidance |
| 📖 **Life Lessons** | Wisdom from experience |
| 💭 **Philosophy** | Deep philosophical insights |
| ⚡ **Quick Wisdom** | Short punchy quotes |

## Installation

```bash
/skill install kingbit-mentor
```

## Usage

### Get Wisdom
```
/mentor               Get random wisdom
/mentor morning      Morning motivation
/mentor evening      Evening reflection
/mentor advice       General advice
/mentor philosophy  Deep thought
/mentor quote        Quick quote
```

### Context-Based
```
/mentor for work     Career advice
/mentor for life     Life guidance
/mentor for coding  Developer wisdom
/mentor for love    Relationship advice
```

### Specific Situations
```
/mentor I'm sad      Comforting wisdom
/mentor I'm confused Clarity advice
/mentor I failed     Resilience wisdom
/mentor I succeeded  Grounding advice
```

## Examples

```
You: /mentor
Kingbit: "The bamboo that bends survives the storm that breaks the oak. Be flexible in your approach, firm in your values."

You: /mentor I'm stressed
Kingbit: "Sit by the river. The debris that blocks your path will float past. Your clarity will return."

You: /mentor for work
Kingbit: "The master carpenter doesn't blame his tools. He sharpens them. Skill is what you do when you can't blame anyone else."
```

## Wisdom Categories

- **Strength**: Perseverance through hardship
- **Clarity**: Finding focus in chaos  
- **Patience**: Timing and waiting
- **Growth**: Learning and improving
- **Balance**: Work and life harmony
- **Purpose**: Finding meaning

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `style` | wise | How wisdom is delivered |
| `length` | medium | Short, medium, or long |

## Uninstall

```bash
/skill remove kingbit-mentor
```
